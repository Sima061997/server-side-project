Movie App

This Web Application provides users with access to information about different movies, their directors and genres. Users are able to sign up, update their personal information and create a list of their own favourite movies.

Request URL HTTP Method Request body data format Response body data format
Get to Database / GET None A text message indicating Database is open
Get a list of movies /movies GET None A JSON object holding data about all the movies
Get data (description, genre, director, image URL, whether it’s featured or not) about a single movie by title /movies/:Title GET None A JSON object holding data about a single movie. Example
{

"Title": "Harry Potter and the Philosopher's Stone",

"Description": "...."

"Genre": {

"Name": "Fantasy",

"Description": "" }

"Director": {

"Name": "Chris Columbus",

"Bio": "",

"Birth": "September 10, 1958" },

"ImageURL": ""
}

"Featured":

"Actors: [ ] "
Get data about a genre (description) by name/title (e.g., “Fantasy”) /movies/genre/:Name GET None A JSON object holding data about a genre. Example
{

"Genre": {

"Name": "Fantasy",

"Description": ""
}

Get data about a director (name, birth year) by name /movies/director/:Name GET None A JSON object holding data about a director. Example:

"Director":
{

"Name": "Olivier Nakache",

"Bio": "",

"Birth": "15 April 1973"
}

Add a new User /users POST A JSON object holding data about the user to add, structured like:

{

"name": "Nemo",

"favoriteMovies": []

}

 A JSON object holding data about the user that was added, including an ID:

{

"id": 1,

"name": "Nemo",

"favoriteMovies": []

}
Update the info (username) about the user /users/:id PUT A JSON object holding data about the user,structured like:

{

"name": "Sam Updated Name",

"favoriteMovies": []

}

 A JSON object holding data about the user including an ID:

{

"id": 1,

"name": "Sam Updated Name",

"favoriteMovies": []

}
Add a movie to the user's list of favorites /users/:id/:movieTitle POST None A text message indicating that a movie has been added to given id
Delete a movie to the user's list of favorites /users/:id/:movieTitle DELETE None A text message indicating that a movie has been deleted to given id
Allow a user to deregister according to id /users/:id DELETE None A text message indicating that an id been deleted
User Login /login POST A JSON object holding username and Password:

{

"Name":
